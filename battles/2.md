# Battle n.1 - Target n.2 - Carrom

[Here's the challenge](https://cssbattle.dev/play/2)

Double solution here, with flex and with grid.
With flex, margin is required to properly align squares. With grid, no margin is needed as we just "turn on" the squares, but is more verbose.

With flex:

```html
<div class="flex-r">
  <div class="flex-c">
    <div class="cell"></div>
    <div class="cell"></div>
  </div>
  <div class="flex-c">
    <div class="cell"></div>
    <div class="cell"></div>
  </div>
</div>
<style>
  body {
    margin: 0;
    height: 300px;
    background: #62374e;
  }
  .flex-c {
    display: flex;
    flex-direction: column;
    justify-content: space-between;
  }
  .flex-r {
    display: flex;
    justify-content: space-between;
    height: 100%;
  }
  .cell {
    width: 50px;
    aspect-ratio: 1;
    background: #fdc57b;
    margin: 50px;
  }
</style>
```
