I wanted to try out [`polygon`](https://developer.mozilla.org/en-US/docs/Web/CSS/basic-shape/polygon) function!

```html
<div></div>
<style>
  body {
    margin: 0;
    background: #F3AC3C;
  }
  div {
    height: 100%;
    clip-path: polygon(140px 0, 100% 0, 100% 100%, 0 100%, 0 140px);
    background: #0B2429;
  }
</style>
```
